# 1-1, 1-2, 1-3

- Relational Database RDB since 1969
- SQL:
  - Structured Query Languragae
- RDBMS:
  - Relational Database Management System
- RDBMS:
  - システム構成はクライアント・サーバ型（C/S 型）
- DDL:
  - CREATE
  - DROP
  - ALTER
- DML:
  - SELECT
  - INSERT
  - UPDATE
  - DELETE
- DCL (Data Control Language):
  - COMMIT
  - ROLLBACK
  - GRANT
    - ユーザーに操作の権限を与える
  - REVOKE
    - ユーザーから操作の権限を奪う
- 定数:
  - SQL に直に書く文字列や数値のこと
  - ’’
    - シングルクオートで表現する
